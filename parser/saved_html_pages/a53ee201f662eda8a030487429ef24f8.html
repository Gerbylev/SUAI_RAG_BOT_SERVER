О сайте

Для просмотра сайта ГУАП настоятельно рекомендуем использовать актуальные версии интернет-браузеров!

Приказ ГУАП от 23.03.2023 № 05-145/23 «Об утверждении и введении в действие Положения об официальном сайте ГУАП»

Доступ к сайту ГУАП может осуществляться с использованием современных интернет-браузеров, поддерживающих спецификацию HTML5. Работоспособность сайта постоянно тестируется на браузерах: Firefox, Google Chrome, Safari, Opera, Brave, Microsoft Edge.

Информация об использовании сайта слабовидящими пользователями и людьми с ограниченными возможностями здоровья

Применение языка разметки HTML5 и адаптивного дизайна обеспечивает эффективный доступ к материалам сайта для людей с ограниченными возможностями здоровья с помощью встроенных средств современных интернет-браузеров.

Полезные комбинации клавиш для улучшения читаемости сайта:

Ctrl + +
— увеличить размер текста
Ctrl + −
— уменьшить размер текста
Ctrl + 0
— установить размер текста в 100%

Сайт ГУАП создается с применением адаптивного дизайна. Эта технология обеспечивает качественное отображение информации на устройствах с различным размером и разрешением экрана (от мобильного телефона до презентационных комплексов высокого разрешения).

Мы постоянно работаем над совершенствованием сайта ГУАП и стремимся использовать передовые решения, обеспечивающие доступность контента для пользователей. Если у вас возникли трудности с доступом к информации на сайте, просим сообщить нам об этом.

Нормативные документы и рекомендации, которыми мы руководствуемся при создании сайта ГУАП:

  • Методические рекомендации представления информации об образовательной организации высшего образования в открытых источниках с учетом соблюдения требований законодательства в сфере образования
  • Приказ Министерства связи и массовых коммуникаций Российской Федерации (Минкомсвязь России) от 30.11.2015 № 483 «Об установлении Порядка обеспечения условий доступности для инвалидов по зрению официальных сайтов федеральных органов государственной власти, органов государственной власти субъектов Российской Федерации и органов местного самоуправления в сети Интернет»
  • ГОСТ Р 52872-2019 «Интернет-ресурсы и другая информация, представленная в электронно-цифровой форме. Приложения для стационарных и мобильных устройств, иные пользовательские интерфейсы. Требования доступности для людей с инвалидностью и других лиц с ограничениями жизнедеятельности»
  • Руководстве по обеспечению доступности веб-контента (Web Content Accessibility Guidelines, WCAG) в редакции 2.1, разработанного Консорциумом Всемирной паутины (World Wide Web Consortium, W3C). Русский перевод:
  • Руководство по созданию сайтов и контента для людей с ограниченными возможностями Фонда «СО-ЕДИНЕНИЕ»

Разработка и сопровождение сайта университета осуществляется управлением цифрового развития ГУАП.
